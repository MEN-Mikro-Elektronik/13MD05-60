Detailed Description

Parameter group handler for MENMON detected parameters.

Author:
klaus.popp@men.de
Date
2007/03/07 11:54:31
Revision
1.9
Project:
SYSPARAM library
This file has the following function groups:
  • Handler for MMPARA parameter group
  • Functions for tuple strings (such as Menmon parameter string)
  • Function to rebuild MENMON parameter string

MMPARA group:

All parameters are read only for user. Firmware can update parameters at any time.

Parameters are non-persistant and cannot be set to defaults.

Preprocessor Switches:
none

int SYSP_MmParaStringBuild (  char *  mmParaStr,
int  mmParaStrMaxLen,
SYSP_SUPER *  super
) 
 

Build MENMON parameter string.

Builds a tuple string from all parameters in all groups of all NV storage objects, whose attributes report that they want to be included in MM parameter string, and builds a tuple string into mmParaStr.

The official parameter names with prefix will be written to MM para string

If a parameter tuple doesn't fit as a whole into mmParaStr, the tuple is omitted and the function returns SYSPARAM_NO_RESOURCE.

Parameters:
mmParaStr OUT receives the created MM parameter string
mmParaStrMaxLen IN max length of mmParaStr (incl. term. 0)
super IN SYSPARAM super object
Returns:
0 on success, or sysparam error code

int SYSP_TusParamSet (  char *  tus,
int  tusMaxLen,
const char *  parName,
const char *  parVal
) 
 

Set value of parameter parName in tuple string.

  • If parName doesn't exist in tuple string, it is created
  • If parVal is NULL or an empty string, parName is deleted from tuple string

Before the value is modified, it is checked wether it would fit in the tuple string. If it would not fit, no modification is done and SYSPARAM_VAL_TRUNCATED is returned.

Parameters:
tus IN tuple parameter string
tusMaxLen IN max length of tuple string (incl. term. 0)
parName IN parameter name to search for
parVal IN new parameter value (can be NULL or empty string)
Returns:
0 if parameter found, or
  • SYSPARAM_NO_RESOURCE - new value doesn't fit into tus
  • SYSPARAM_INVAL_VALUE - invalid characters in parVal
