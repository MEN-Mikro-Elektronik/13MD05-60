int Prod2ToRawDate (  void *  dest,
const char *  src
)  [static]
 

converts string to raw date value

String format must be in US notation m/d/y

Where m is the month 1..12 (leading zeroes allowed) Where d is the day of month 1..31 (leading zeroes allowed) Where y is the year. If no century included, 2000 is added to year.

An empty src string sets the raw date to 0xffff (invalid)

Includes the validation function returns SYSPARAM_INVAL_VALUE if invalid date

Probe for EEPROD2 parameter group.

On success, (and if hwName matched), the created parameter group is added to the NVS' list.

Parameters:
nvs IN NV storage object (fully initialized)
expHwNames IN array of bord names tolerated (null-terminated). Can be NULL, in this case, any name will match.
ignoreNvsError IN if TRUE, a NVS read error is ignored
Returns:
sysparam error code
