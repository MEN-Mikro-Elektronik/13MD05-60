SYSPARAM system dependent includes/defines.

Preprocessor Switches:
  • SYSP_CONFIG_MDIS_BUILD - Set this when compiling within MDIS build environment.

  • DBG - Compile with debug messages enabled (very few are there)
  • SYSP_CONFIG_UNITTEST - include unittests

If SYSP_CONFIG_MDIS_BUILD is not set, the following switches can be set:

  • SYSP_CONFIG_LINUX, SYSP_CONFIG_MENMON, SYSP_CONFIG_VXWORKS, SYSP_CONFIG_OS9 - the operating system used.
  • SYSP_CONFIG_LE, SYSP_CONFIG_BE little or big endian
