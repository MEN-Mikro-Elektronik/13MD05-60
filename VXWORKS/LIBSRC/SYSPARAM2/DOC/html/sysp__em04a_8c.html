int SYSP_NvsEm04ACpuProbe (  SYSP_MMSMB_SUPER *  xSuper  )  [static]
 

Probe for EM04A CPU EEPROM and non-persistant mmpara section.

Creates all parameter sections that shall be present in EM04A CPU EEPROM even if there is an EEPROM error.

Also creates the MENMON parameter section "mmpara"

sysparam error code if fatal error
