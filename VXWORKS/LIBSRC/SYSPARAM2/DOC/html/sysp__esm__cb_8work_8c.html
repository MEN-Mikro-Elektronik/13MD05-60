int SYSP_EsmCarrierProbe (  SYSP_SUPER *  super,
int  smbBusNo
) 
 

Probe for all possible known ESM carrier boards.

Prefix will be 'c' always. SMB addresses are hardcoded (0xac).

If nothing could be read from SMB (no board EEPROM there), no parameter group is created and SYSPARAM_EEPROM_ERROR is returned.

If an unknown/unprogrammed board is found, a parameter group for standard EEPROD section is setup

Parameters:
super IN sysparam super object
smbBusNo IN SMB bus number
Returns:
sysparam error code
