void* SYSP_Alloc (  SYSP_ALLOC *  alloc,
int  size
) 
 

Allocate block from heap specified with alloc.

Tries to allocate size bytes from the heap maintained with alloc. On Success, returns a pointer to the first byte of the block. The data block is filled with zeroes.

If not enough mem available, NULL is returned.

Warning:
This is not a full malloc() implementation. A memory block can only be free'ed when no other allocs have been made in the meantime; i.e. it works like a stack.

Create a parameter group object for VxWorks bootline parameters.

On success, the parameter group is added to the list of nvs groups

If the passed vxBline is an empty string, the vxBline is read from the non volatile storage. If there is a physical read error, or if it contains non-printable chars, default bootline is setup.

If the passed vxBline is not-empty, it is checked for non-printable chars, and defaults are applied if so.

Parameters:
nvs IN NV storage object (fully initialized)
nvsOffset IN starting byte offset of vxbline within NV storage
nvsSize IN size of vxbline in NV storage in bytes
vxBline IN the vxworks boot line in memory
vxDefBline IN VxWorks boot line with defaults, if NULL, an empty bootline will be the default
provideKerpar IN if SYSP_PROVIDE_KERPAR, let "kerpar" be an alias to "s"
Returns:
sysparam error code
