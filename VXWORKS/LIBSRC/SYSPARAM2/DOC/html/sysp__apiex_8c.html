int SYSP_MenBrdNameWithModel (  char  prefix,
char *  buf
) 
 

Determine chars 3..9 (e.g.

EM04-00) of MEN product name

Uses SYSP_brd and SYSP_brdmod parameter. SYSP_brd value is padded with '-' if it contains only 4 chars

If SYSP_brd not valid, return empty string. If SYSP_brdmod not valid, model number will be missing from string

Parameters:
prefix IN board prefix (e.g. 'c') (0 if none)
buf OUT filled with name (must have space for 7 chars + EOS)
Returns:
sysparam error code
