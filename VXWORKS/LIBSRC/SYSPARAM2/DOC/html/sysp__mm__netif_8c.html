Detailed Description

Special fake NVS section/parameter group that provides the nmacX and nspeedX parameters of network interfaces that store those values into their own SROM.

Author:
klaus.popp@men.de
Date
2005/06/23 15:54:46
Revision
2.5
Project:
SYSPARAM library

Query NETIF subsystem for known devices that provide their own EEPROM

If any NETIF device found, create a NVS_OBJECT that handles ALL those devices.

For each device, create one parameter group

Preprocessor Switches:
none
