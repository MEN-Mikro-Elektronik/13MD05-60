const char *  defVal
) 
 

Create a parameter group object for simple strings.

On success, the parameter group is added to the list of nvs groups

The string is read from the non volatile storage. If there is a physical read error, or if it contains non-printable chars, default string is setup.

Parameters:
nvs IN NV storage object (fully initialized)
name IN name of parameter group and name of the parameter itsself
alias IN alias name of parameter (can be NULL)
nvsOffset IN starting byte offset of string within NV storage
nvsSize IN max size of string in NV storage in bytes
defVal IN default value for string (can be NULL)
Returns:
sysparam error code
