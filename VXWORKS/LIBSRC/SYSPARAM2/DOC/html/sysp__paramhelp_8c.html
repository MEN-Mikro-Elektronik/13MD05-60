const SYSP_HELP_DESC* FindDescEntry (  const char *  parName  )  [static]
 

Find entry in desc table according to parName.

If parName ends with a digit, it matches any entry that similar entries (e.g. parName=nspeed2 matches entry nspeed0), however this works only with official, not alias names.

Parameters:
parName IN parameter name to search for
Returns:
entry in desc table or NULL if not found
