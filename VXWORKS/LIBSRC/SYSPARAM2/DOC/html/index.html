SYSPARAM2 Documentation

for SYSPARAM2 fileset 1.30

The system parameter library provides a generic way to access parameters, such as

  • parameters stored in non volatile storage (NVS)
    • board inventory parameters (production data)
    • configurable parameters (e.g. console baudrate)
  • parameters auto-detected by firmware

User callable routines

  • SysParamGet()
  • SysParamSet()
  • SysParamSetEx()
  • SysParamEnum()
  • SysParamGetAll()
  • SysParamSetDefaults()
  • SysParamMakePersistant()
  • SysParamErrStr()

Before any of these functions can be called, a system specific init routine must be called, e.g. Em04SysParamInit().
